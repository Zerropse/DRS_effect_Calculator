# 🏎️ DRS Effectiveness & Overtake Simulator

A physics-based motorsport simulation tool that analyzes the impact of DRS (Drag Reduction System) and predicts overtaking opportunities.

## 🚀 Features

- Aerodynamic drag-based speed simulation
- DRS performance comparison
- Time saved & efficiency calculation
- Overtake prediction model
- Interactive graphs (speed, distance, gap)

## 📊 Tech Stack

- Python
- Streamlit
- NumPy
- Matplotlib

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
